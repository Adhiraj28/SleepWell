# SleepWell

Android sleep tracking app built with Java for COMP-2560 Mobile Application Development.

## Features
- Log daily sleep entries with bedtime, wake time, duration, and quality rating
- View sleep history with RecyclerView and CardView
- Dashboard with stats (total entries, average hours, goal progress)
- User settings with SharedPreferences
- Room database with LiveData and ViewModel architecture
- Fragment-based dashboard UI
